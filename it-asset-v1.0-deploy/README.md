# 企业运维管理系统

基于 Flask + SQLite 的轻量级运维管理系统，适用于内网部署。

## 功能模块

### 资产管理
- 资产台账管理（增删改查）
- 资产分类管理
- 资产全生命周期（采购→入库→分配→调拨→维修→报废）
- 软件授权管理
- 合同与供应商管理
- 盘点管理
- 到期预警
- 报表统计

### 运维管理
- **设备台账** — 记录负责设备（网络、安全、服务器、存储等）的详细参数
- **每日巡检** — 工程师按设备逐项打勾，记录状态，异常自动带出处理动作
- **工作日志** — 每位工程师每天填写做了什么（不限于巡检，包含维修、变更、沟通等）
- **关联关系** — 工作日志可引用设备编号或巡检记录，形成完整链路

### 报表管理
- **日报管理** — 按日汇总工作
- **周报管理** — 按周汇总工作
- **月报管理** — 按月汇总工作

### 用户管理
- 管理员 admin 账户（默认密码 admin）
- 管理员可新增用户，对所有模块进行授权
- 默认用户仅有日报模块权限

## 技术栈

- 后端：Python Flask
- 数据库：SQLite
- 前端：Bootstrap 5 + jQuery
- 部署：单机内网

## 快速启动

```bash
pip install -r requirements.txt
python app.py
```

访问 http://服务器IP:5003

## 默认账号

- 管理员：`admin` / `admin`
- 新用户默认密码：`123456`
